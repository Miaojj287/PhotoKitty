# 📸 PhotoKitty - 献给一位特别的人

> 一份用代码编织的生日礼物，记载着我们一起走过的每一个瞬间。

这不仅仅是一个 3D 照片墙，更是一个承载回忆的时光胶囊。每一张悬浮的照片，都是我们共同书写的故事；每一个闪烁的光点，都代表着那些美好的时刻。

通过 AI 手势识别，你可以用手掌控制这些回忆——让它们散开成漫天星辰，或聚合成完整的画面。就像我们的故事，无论怎样变换形态，都始终紧密相连。

**生日快乐！希望这份礼物能让你感受到这一年我们一起创造的美好点滴。**

## 💌 如果你想添加新照片...

### 步骤 1：准备照片
将你喜欢的照片按照 `1.jpg`, `2.jpg`, `3.jpg` ... 的格式命名，建议使用4:3比例。

在一个个编辑照片的时候我发现这个比例还挺有意思的，如果两个人贴的不够近的照片是用不了的，怎么不算是一种浪漫呢。

### 步骤 2：放入文件夹
将照片放到 `public/photos/` 文件夹中

### 步骤 3：修改数量
打开 `src/App.tsx`，找到第 15 行：
```typescript
const TOTAL_NUMBERED_PHOTOS = 52; // 改成你的照片数量
```

### 步骤 4：重新构建
```bash
npm install
npm run build
```

## 📄 技术细节

### 项目结构
```
PhotoKitty/
├── src/
│   ├── App.tsx          # 主应用组件（3D场景、手势控制）
│   ├── App.css          # 响应式样式
│   └── main.tsx         # 应用入口
├── public/
│   ├── photos/          # 照片文件夹（1.jpg - 52.jpg）
│   └── fonts/           # 3D文字字体
└── dist/                # 构建输出目录
```

### 关键配置
- **基础路径**：`/PhotoKitty/` (GitHub Pages)
- **照片数量**：52 张
- **粒子数量**：52,000 个
- **目标帧率**：60 FPS

### 浏览器要求
- 支持 WebGL 的现代浏览器
- 建议使用 Chrome、Safari、Edge 或 Firefox 最新版
- 需要摄像头权限以使用手势控制功能
